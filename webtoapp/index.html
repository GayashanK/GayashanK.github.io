A) Open In XCode - top

This template was developed using Swift 5.2 which is the latest version of the Swift. Minimum compatible XCode version is Version 11.6 (11E708).>

Once you open project you will see following file structure in the image./p>

  1. Create bundle itentifire in your developer account and change current bundle identifire(com.sfdigitals.Product) to it.
  2. Use your apple developer account provitioning profile to signing project.
  3. Setup firebase account and add Google-Info.plist
  4. Add APNs AuthKey form your developer account to firebase project

B) Setup Configuration File - top

Open Configurations.swift file using XCode

  1. Please edit Configurations.swift file according to the comments in file.
  2. Add your logo in Image.assets to branding application.
  3. Add your app icon to Image.assets.
  4. Run the project on Simulator or in Real Device using XCode
